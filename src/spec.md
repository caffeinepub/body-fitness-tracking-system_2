# Specification

## Summary
**Goal:** Build a Body Fitness Tracking System website with user dashboard, BMI calculator, and progress tracking features.

**Planned changes:**
- Create Home page with hero section, navigation bar (Home, Dashboard, Features, BMI Calculator, About, Contact), and "Get Started" button
- Build User Dashboard with step counter, water intake tracker, calories burned display, BMI summary, weekly progress chart, date selection, and reset button
- Implement BMI Calculator page with height/weight inputs and category display
- Add Features page listing system capabilities
- Create About page with project description, objectives, technologies, and team section
- Build Contact page with form (name, email, message) and college details
- Add footer to all pages displaying "BYK College of Commerce", team members, and academic year "2025-26"
- Implement frontend-only login/register system using local storage
- Add dark mode toggle with theme persistence
- Design with fitness-themed styling using Tailwind CSS, card layouts, gradient buttons, and modern fonts (Poppins/Roboto)
- Ensure responsive design for desktop, tablet, and mobile
- Add animations and hover effects to interactive elements
- Store all tracking data in Motoko backend actor for persistence

**User-visible outcome:** Users can register/login, track daily steps and water intake, calculate BMI, view weekly progress charts, and monitor fitness metrics across all devices with data persistence.
